🧠 KAI: 키스트로크 데이터 기반 멀티모달 심리 상담 LLM
⸻

📌 프로젝트 소개
사용자가 텍스트를 입력하는 과정에서 발생하는 타이핑 데이터(타수, 입력 속도 등)를 분석하여 감정 상태를 추론하고, 이를 기반으로 맞춤형 공감 대화를 제공하는 AI 심리 상담 플랫폼입니다.
최종 전송된 텍스트 내용만으로는 파악하기 어려운 내담자의 내면 상태를 입력 과정의 물리적 지표를 통해 보완적으로 분석하는 멀티모달 접근 방식을 취합니다.

⸻

✨ 주요 기능
1. 실시간 키스트로크 데이터 수집

사용자가 채팅창에 메시지를 입력하는 동안 발생하는 키 입력 횟수(Key Count)와 입력 소요 시간(Duration)을 측정합니다.

이를 기반으로 초당 타수(Speed)를 계산합니다.

2. 타이핑 속도 기반 감정 상태 판별
계산된 타이핑 속도를 기준으로 사용자의 현재 감정 상태를 3단계로 1차 분류합니다.

Speed > 5: 불안함 (과도하게 빠른 입력)

Speed < 1: 무기력함 (현저히 느린 입력)

그 외: 안정적

3. LLM 기반 맞춤형 심리 상담 제공

Anthropic Claude 모델에 심리 상담사 페르소나를 부여하여 대화를 진행합니다.

전송된 텍스트 내용과 함께 앞서 추론된 '감정 상태' 데이터를 프롬프트에 주입하여, 현재 감정에 최적화된 공감적 답변을 생성합니다.

⸻

⚙️ 핵심 동작 방식
사용자가 웹 프론트엔드(index.html)에서 텍스트를 입력.

JavaScript가 입력 시작 시간과 누적 키 입력 횟수를 추적.

'전송' 버튼 클릭 시, 텍스트(text)와 타이핑 속도(speed)를 FastAPI 백엔드(/analyze)로 POST 요청.

백엔드에서 속도 값을 평가하여 감정(emotion) 상태 확정.

사용자 상태: {emotion}. 내용: {text} 형태로 프롬프트를 구성하여 Claude API 호출.

반환된 LLM의 상담 답변을 프론트엔드 채팅 UI에 출력 및 스크롤 자동 이동.

⸻

▶ 실행 방법
1. 사전 요구사항 설치
Python 환경에서 아래 명령어를 통해 필요한 패키지를 설치합니다.

Bash
pip install fastapi uvicorn anthropic
2. API 키 확인
main.py 코드 내 client = anthropic.Anthropic(api_key="...") 부분에 유효한 API 토큰이 입력되어 있는지 확인합니다.

3. 서버 실행
main.py와 index.html이 동일한 디렉토리에 있는 상태에서 터미널을 열고 다음 명령어를 실행합니다.

Bash
uvicorn main:app --reload
4. 서비스 접속
서버가 정상적으로 실행되면 웹 브라우저 주소창에 http://127.0.0.1:8000을 입력하여 상담 인터페이스에 접속합니다.

⸻

🚀 향후 개선 과제
멀티모달 기능 완성: 기획 의도에 포함되었던 안면 표정 분석을 위해 MediaPipe 등의 영상 비전 프레임워크를 추가 연동하여 감정 분석의 정확도 향상.

보안 강화: 소스 코드 내에 하드코딩된 API Key를 .env 파일로 분리하여 환경 변수로 관리.

상담 맥락 유지: 현재 단발성으로 처리되는 API 호출 구조를 개선하여, 이전 대화 기록을 누적하고 문맥을 기억하는 기능 추가.
