# 🧠 KAI: 키스트로크 데이터 기반 멀티모달 심리 상담 LLM

---

## 📌 프로젝트 소개
사용자가 텍스트를 입력하는 과정에서 발생하는 타이핑 데이터(타수, 입력 속도 등)를 분석하여 감정 상태를 추론하고, 이를 기반으로 맞춤형 공감 대화를 제공하는 AI 심리 상담 플랫폼입니다. 

최종 전송된 텍스트 내용만으로는 파악하기 어려운 내담자의 내면 상태를 입력 과정의 물리적 지표를 통해 보완적으로 분석하는 멀티모달 접근 방식을 취합니다.

---

## ✨ 주요 기능

**1. 실시간 키스트로크 데이터 수집**
* 사용자가 채팅창에 메시지를 입력하는 동안 발생하는 키 입력 횟수(Key Count)와 입력 소요 시간(Duration)을 측정합니다.
* 이를 기반으로 초당 타수(Speed)를 계산합니다.

**2. 타이핑 속도 기반 감정 상태 판별**
계산된 타이핑 속도를 기준으로 사용자의 현재 감정 상태를 3단계로 1차 분류합니다.
* `Speed > 5`: **불안함** (과도하게 빠른 입력)
* `Speed < 1`: **무기력함** (현저히 느린 입력)
* `그 외`: **안정적**

**3. LLM 기반 맞춤형 심리 상담 제공**
* Anthropic Claude 모델에 심리 상담사 페르소나를 부여하여 대화를 진행합니다.
* 전송된 텍스트 내용과 함께 앞서 추론된 '감정 상태' 데이터를 프롬프트에 주입하여, 현재 감정에 최적화된 공감적 답변을 생성합니다.

---

## ⚙️ 핵심 동작 방식
1. 사용자가 웹 프론트엔드(`index.html`)에서 텍스트를 입력합니다.
2. JavaScript가 입력 시작 시간과 누적 키 입력 횟수를 추적합니다.
3. '전송' 버튼 클릭 시, 텍스트(`text`)와 타이핑 속도(`speed`)를 FastAPI 백엔드(`/analyze`)로 `POST` 요청합니다.
4. 백엔드에서 속도 값을 평가하여 감정(`emotion`) 상태를 확정합니다.
5. `사용자 상태: {emotion}. 내용: {text}` 형태로 프롬프트를 구성하여 Claude API를 호출합니다.
6. 반환된 LLM의 상담 답변을 프론트엔드 채팅 UI에 출력하고 스크롤을 자동으로 이동시킵니다.
