# 키스트로크 과정 데이터 기반 멀티모달 심리 상담 LLM

---

## 개요 및 수정사항


인공지능 상담 서비스는 많습니다. 하지만 우리는 대화의 '내용'에만 집중해 왔습니다. 사용자가 입력을 멈추거나, 서둘러 타자를 칠 때의 그 '호흡' 속에 담긴 망설임과 불안을 읽어낼 수는 없을까? KAI는 이 질문에서 시작된 프로젝트입니다.텍 스트 데이터에 타이핑 패턴(속도)이라는 생체 데이터를 결합했습니다. 사용자가 타이핑하는 속도가 빠르면 불안함으로, 너무 느리면 무기력함으로 감지하여 AI가 그에 맞는 온도의 답변을 내놓도록 설계했습니다.

즉, 사용자가 텍스트를 입력하는 과정에서 발생하는 데이터(입력 지연, 삭제, 커서 움직임 등)와 안면 표정(영상)을 동시에 분석하여
최종 전송된 텍스트만으로는 포착하기 어려운 내면의 감정 상태를 추론하고 공감적으로 반응하는 상담용 LLM 플랫폼 개발하고자 하였습니다.

## 실행 방법

다음과 같은 파이썬 모듈을 요구합니다.

```fastapi uvicorn anthropic```

1. 깃허브의 모든 파일(main.py, index.html)을 한 폴더에 다운받는다.
2. 해당 폴더에서 CMD를 실행한다.
3. ```uvicorn main:app --reload``` 의 코드를 실행한다.
4. 잠시 후 http://127.0.0.1:8000이라는 주소가 보이면 이를 복사하여 웹 브라우저의 주소창에 붙여넣는다.
5. 잠시 후, 상담 창이 나타나며 프로그램이 실행된다.
