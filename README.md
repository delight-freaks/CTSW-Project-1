# 키스트로크 과정 데이터 기반 멀티모달 심리 상담 LLM

---

## 개요 및 수정사항

사용자가 텍스트를 입력하는 과정에서 발생하는 데이터(입력 지연, 삭제, 커서 움직임 등)와 안면 표정(영상)을 동시에 분석하여
최종 전송된 텍스트만으로는 포착하기 어려운 내면의 감정 상태를 추론하고 공감적으로 반응하는 상담용 LLM 플랫폼 개발.

-> (2026.06.11 추가) 교수님께서 단독적인 재현 가능성의 여부를 언급하셔서, 해당 프로젝트의 초기 Prototype을 첨부하는 것으로 전면 수정하였습니다.
   기존 "(키스트로크 + 비전 + 침묵 감지 + AI 상담)"의 모듈에서 초기 개발할때 생성했던 프로토타입의 기능인
   현재 "(경량화 키스트로크 + AI 상담)" 기능만이 구현된 상태입니다.
   
   교수님의 과제 제출 의도에 부합하지 않았던, 일종의 뽐내기 식의 발표였던 것 같아, 굉장히 죄송하다는 말씀을 드리고 싶습니다.

1. **키스트로크 패턴** 사용자의 타자 패턴을 감지하여 타자 패턴에 따른 심리 상태를 판정하여 AI에 전달, 이를 기반으로 더 깊은 상담을 유도함.
2. **AI 상담**  사용자와 대화를 주고받으며, 사용자의 감정 해소를 위한 상담자의 역할을 하게 된다.
