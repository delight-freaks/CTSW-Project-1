# 키스트로크 과정 데이터 기반 멀티모달 심리 상담 LLM

## 1. 무엇을 하는 프로그램인가요?
사용자가 텍스트를 입력하는 과정에서 발생하는 데이터(입력 지연, 삭제, 커서 움직임 등)와 안면 표정(영상)을 동시에 분석하여
최종 전송된 텍스트만으로는 포착하기 어려운 내면의 감정 상태를 추론하고 공감적으로 반응하는 상담용 LLM 플랫폼 개발하고자 하였습니다.

---

## 2. 핵심 기능 (Features)
- **키스트로크 감지**: 사용자의 타이핑 패턴을 분석하여 심리 상태를 유추
- **AI 기반 심리 상담**: 웹으로 연동된 AI가 사용자와 내담
- **프롬프트 엔지니어링**: 상담 AI에 맞게, 상담이 잘못된 방향으로 가지 않도록 검열

## 3. 실행 방법
### 사전 요구사항
```fastapi uvicorn anthropic```

1. 깃허브의 모든 파일(main.py, index.html)을 한 폴더에 다운받는다.
2. 해당 폴더에서 CMD를 실행한다.
3. ```uvicorn main:app --reload``` 의 코드를 실행한다.
4. 잠시 후 http://127.0.0.1:8000이라는 주소가 보이면 이를 복사하여 웹 브라우저의 주소창에 붙여넣는다.
5. 잠시 후, 상담 창이 나타나며 프로그램이 실행된다.
